First principles

The rules that keep it consistent

If you remember nothing else, remember these.

  1. One stylesheet, copied verbatim. The <style> block + icon sprite above are identical on every page. Never hand-edit per page.
  2. Tokens only. Colours, fonts, --radius, --shadow and the spacing scale come from variables. Don't introduce new hex colours or arbitrary px values.
  3. Components over prose. Lead with cards, stats, notes, steps, grids and tracks. Keep paragraphs short; let the components carry the information.
  4. Re-theme by variable, not by rewrite. A new topic changes the accent variables (and the content) — nothing else.
  5. Relative links. Always item-2.html, never /item-2.html, so it works under username.github.io/REPO/.

Component gallery

Cards, stats & the icon chip

The workhorses. A plain card, a titled card with an icon chip and mono sub-label, and a category card with a coloured left bar.

Titled card

.card · .card-title · .ico · .val

Use for any self-contained block. The 42px .ico chip + Fraunces h3 + mono .val sub-label is the standard header.

Category card

.card.cat · set --bc

Add class cat and set --bc to a category accent for a coloured left bar + tinted icon. Use for grouped content (regions, weeks, body-areas).

Re-theme

One project, one accent — swap these variables

To make this system fit a new topic, change only the [THEME] block in :root. Everything else (components, layout, scale) stays identical.

--primary#1E5945
--primary-deep#143E30
--primary-soft#E7F0EA
--good#2E8B6B
--watch#B7740E
--alert#BC4B45

Travel theme — warm terracotta

:root{ --bg:#F7F4EF; --ink:#241C16; --ink-soft:#5A5048; --ink-faint:#8A7F74; --line:#E7E1D8; --line-strong:#D8D0C4; --card:#FFFFFF; --primary:#B0532E; --primary-deep:#7E3A1F; --primary-soft:#F4E7DE; --glow:rgba(176,83,46,.06); }

Physiotherapy theme — calm teal

:root{ --bg:#F2F5F7; --ink:#15232B; --ink-soft:#46545C; --ink-faint:#7A878F; --line:#DEE6EA; --line-strong:#CDD8DD; --card:#FFFFFF; --primary:#1E6E78; --primary-deep:#0F4A52; --primary-soft:#E0F0F1; --glow:rgba(30,110,120,.06); }
Status colours (--good/--watch/--alert) and the category accents usually stay put — they read the same in any theme. Only retune them if they clash with a very different --primary.
